# -Calculator
        📌 Project Description: Calculator using HTML, CSS & JavaScript
This project is a basic web calculator that performs simple arithmetic operations like:
        Addition (+)
        Subtraction (−)
        Multiplication (×)
        Division (÷)
It is created using:
HTML → To create the structure of the calculator
CSS → To design and style the calculator
JavaScript → To add functionality and perform calculations

How It Works:-
1) When a button is clicked, appendValue() adds the number/operator to the display.
2) When = is pressed, calculate() evaluates the expression.
3) C clears the display.
4) DEL removes the last character.
5) CSS makes the calculator look clean and centered.
   
🎯 Conclusion:-
This simple calculator project helps beginners understand:
    DOM manipulation
    Event handling
    Functions in JavaScript
    Basic web design using CSS Grid
